**Step 1: Update all LogRecord field access sites to use accessors.**

Replace direct field access patterns with accessor function calls. The key transformation:
- `record.level` -> `record.level(record)` -- BUT this is ambiguous. To avoid confusion with the variable name, use the module-qualified form or let binding:
  - In files that import `birch/record`, use `record.level(rec)` where `rec` is the LogRecord variable
  - Watch for variable shadowing: if the LogRecord variable is named `record`, you cannot write `record.level(record)` because `record` refers to the module. Rename the variable to `rec` or `log_record` where needed.

Files to update for LogRecord:
